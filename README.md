## 🚀 技术特性

### REM 适配方案
- **基准**: 1rem = 100px (1920px设计稿)
- **计算**: `font-size: 5.208vw` (100/1920*100)
- **断点**: 6个响应式断点覆盖全设备

### 设计系统
- **主色**: `#00D26A` (品牌绿)
- **字体**: Noto Sans SC + Inter
- **间距**: 8px基准单位系统
- **圆角**: 4px/8px/12px/16px/28px(胶囊)
- **阴影**: 3级阴影系统

### 响应式断点
| 断点 | 目标设备 | HTML font-size |
|------|----------|----------------|
| 1920px+ | 大屏桌面 | 100px |
| 1440px | 标准桌面 | 6.944vw |
| 1280px | 小屏桌面 | 7.813vw |
| 1024px | 平板横屏 | 9.766vw |
| 768px | 平板竖屏 | 13.021vw |
| 480px | 移动端 | 21.333vw |
| 375px | 小屏移动 | 26.667vw |

## 🔧 使用方法

### 本地预览
```bash
# 直接打开
open index.html

# 或使用本地服务器
npx serve .
```

### 图片替换
1. 将新图片放入 `images/` 目录
2. 保持与原图片相同的文件名
3. 刷新页面即可

### 样式调整
- 全局变量: `css/base.css` 中的 `:root`
- 组件样式: `css/style.css` 按区块组织
- 响应式: `css/responsive.css` 按断点组织
